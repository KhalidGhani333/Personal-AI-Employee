# Plan Creator Skill

Create detailed execution plans for complex multi-step tasks.

## Metadata
- **Tier**: Silver
- **Priority**: High
- **Dependencies**: Task Manager, Dashboard Updater
- **Triggers**: Complex task requests, multi-step workflows

## Capabilities
- Analyze complex tasks
- Break down into steps
- Identify dependencies
- Create actionable plans
- Track plan progress
- Support Ralph Wiggum loop

## Usage
```bash
/plan-creator --task="{task_description}"
```

## Expected Input
- Complex task description
- Requirements and constraints
- Success criteria
- Available resources

## Expected Output
- Detailed Plan.md in /Plans folder
- Step-by-step execution checklist
- Dependency mapping
- Progress tracking
