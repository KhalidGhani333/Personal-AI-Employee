# Accounting Integrator Skill

Integrate with Odoo accounting system for financial management.

## Metadata
- **Tier**: Gold
- **Priority**: High
- **Dependencies**: Invoice Generator, Business Auditor
- **Triggers**: Financial transactions, invoice creation

## Capabilities
- Sync transactions to Odoo
- Create invoices in Odoo
- Generate financial reports
- Track expenses
- Manage chart of accounts
- Reconcile bank statements

## Usage
```bash
/accounting-integrator --action={sync|create_invoice|report}
```

## Expected Input
- Transaction data
- Invoice information
- Expense records
- Bank statements

## Expected Output
- Synced data in Odoo
- Financial reports
- Reconciliation status
- Accounting dashboard updates
